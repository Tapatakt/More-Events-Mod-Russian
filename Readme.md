# More Events Mod Russian

Локализация замечательного мода на Stellaris "[More Events Mod](https://steamcommunity.com/sharedfiles/filedetails/?id=727000451)", который добавляет больше аномалий, событий, происхождений и вообще всякого прикольного.

Локализация актуальна для версии мода 2.20 от 15 июня 2016 для Stellaris 4.4

Ещё я сделал и использовал вспомогательный инструмент для перевода модов на игры от Парадоксов, его можно найти [здесь](https://github.com/Tapatakt/ParadoxLocalisationHelper).

[Мод в Steam](https://steamcommunity.com/sharedfiles/filedetails/?id=3688697656)

[Предшествующая заброшенная локализация от knon_20](https://steamcommunity.com/sharedfiles/filedetails/?id=2854635168&tscn=1773961857) (приличная часть текста всё ещё оттуда)

Локализация новых и изменённых с 2024 года строк выполнена при помощи Kimi K2.5 и K2.7 и постепенно совершенствуется вручную.

Обязательно ставьте локализацию в самый конец списка модов.
